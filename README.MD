## 环境
  node
  mongodb

## 启动
```
  启动前端服务
  cd attitudemusic
  npm i
  npm run serve

  启动后端服务
  cd attitudeBackground
  npm i
  nodemon app.js

  启动资源服务
  cd NeteaseCloudMusicApi-master
  npm i
  nodemon app.js
```
